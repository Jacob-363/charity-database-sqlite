# Charity Database (SQLite)

A fictional charity database built with SQLite as a learning project.

## Overview
This project models a relational database for a fictional charity, including donors, volunteers, recipients, payments, and related media.

## Contents
- `schema.sql` – database schema
- `queries.sql` – example queries
- `DESIGN.md` – detailed design decisions and constraints
- Entity Relationship Diagram

## Notes
All data and entities in this project are fictional.
